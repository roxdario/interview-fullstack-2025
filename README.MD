# Area of Impact Dashboard

FullStack application for the technical interview.  
The backend exposes a REST API that reads AOI data from a JSON file, and the React frontend shows a dashboard with one chart and several tables.

## Requirements

- Docker and Docker Compose installed

## Quick start

git clone https://github.com/roxdario/interview-fullstack-2025.git
cd interview-fullstack-2025
docker compose up --build

Then open:

- Backend health: http://localhost:8080/health  
- Dashboard: http://localhost:3000
